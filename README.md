# Student_data_management_dashboard
A Django-based Student Management System for colleges that provides a comprehensive admin dashboard to manage students, teachers, departments, subjects, exams, holidays, events, revenue, and awards. The dashboard is interactive, visually appealing, and responsive.

Dashboard
Welcome Admin section with profile menu

Revenue
Awards
